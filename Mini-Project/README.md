# Pathfinding Visualizer

An interactive web-based pathfinding visualizer that demonstrates how classic graph traversal algorithms work in real time.

Users can create walls, set start and end points, and watch **Breadth-First Search (BFS)** explore the grid and find the shortest path. The visualization highlights visited nodes and animates the final path, making it easy to understand the algorithm step by step.

Built using **HTML, CSS, and JavaScript**, this project focuses on core **Data Structures and Algorithms (DSA)** concepts with an intuitive UI and smooth animations.
